◆ sqrt() [2/3]

REAL iRRAM::sqrt ( const REAL &  x)

Computes the square root of the argument x.

  • For domain \([A;B]\) with \(A>0\), the square root function \(f=\sqrt{}\) has a Lipschitz bound:

    \[ \forall x,y\in[A;B]: |f(x)-f(y)| \leq K\cdot|x-y| = K\cdot|f(x)+f(y)|\cdot|f(x)-f(y)| \Longleftrightarrow 1/|f(x)+f(y)| \leq K \]

    Since \(\max_{x,y \in [A;B]} 1/|f(x)+f(y)| = 1/|2f(A)|\leq K\) for the function's argument \(x\in[x_c\pm x_\varepsilon]\) the error therefore is \(|f(x_c)-f(x)| \leq K|x-x_c| < Kx_\varepsilon \leq x_\varepsilon/(\sqrt{\check x}+\sqrt{\check x-x_\varepsilon}) < x_\varepsilon/(2\sqrt{\check x-x_\varepsilon})\).

    This case, \(x\geq A=\check x-x_\varepsilon>0\), is chosen when \(\check x\geq 4x_\varepsilon\Longrightarrow |x|\geq 3x_\varepsilon\), therefore \(|\sqrt{x_c}-\sqrt x| \leq x_\varepsilon/(2\sqrt{\check x-x_\varepsilon})\). The absolute precision used to compute \(\sqrt{x_c}\) is \(\max\{P,e[z]\}\) where \(z\geq x_\varepsilon/(2\sqrt{\check x-x_\varepsilon})\) and \(e[v]=e\) for any sizetype representation \(m\cdot2^e\) of \(v\). The relative precision is \(\max\{e[\check x]+P,e[z]\}\).

  • Otherwise, \(x=0\) or \(4x_\varepsilon\geq\check x=\hat x-2^e>\hat x\cdot(1-2^{-32})\), then \(\sqrt{|x|} <\sqrt{\hat x+x_\varepsilon} <\sqrt{4x_\varepsilon/(1-2^{-32})+x_\varepsilon} <2.25\sqrt{x_\varepsilon}\) and \(0\approx\sqrt{|x|}\) is chosen as an approximation with error \(\geq3\sqrt{x_\varepsilon}\).
Todo:
For \(0\in[x_c\pm x_\varepsilon]\) use \( \approx[\sqrt{x_\varepsilon}\pm\sqrt{x_\varepsilon}] \) as result to decrease the error and disable representation of non-co-domain values smaller than zero.
Todo:
To determine \(0\notin[x_c\pm x_\varepsilon]\) check \( \check x\geq cx_\varepsilon\) with \(c\in\{2,3\} \) for tighter error bounds in both cases.
Todo:
In case \(0\notin[x_c\pm x_\varepsilon]\), does it make sense to use the bound \( x_\varepsilon/(\sqrt{\check x}+\sqrt{\check x-x_\varepsilon}) \) instead of \(x_\varepsilon/(2\sqrt{\check x-x_\varepsilon})\)?
